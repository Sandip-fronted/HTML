1.WEBSITE

=> a set of related webpages

2.WWW

=> world wide web- The World Wide Web is a subset of the Internet consisting of websites and webpages that are accessible via a web browser.

3.HTTP/HTTPS

=> HTTP(Hyper Text Transfer Protocol)- set of rules
=> HTTPS (hyper text transfer protocol secure)-Transfer data through network OR more secure to http

4.CLIENT SERVER

=> denoting a computer system in which a central server provides data to a number of networked workstations

5.DOMAIN

=> A domain is a network of computer device that are controlled by one set authority and have specific guidelines.

6.HOST

=> A host is a computer or other device that is connected to a network and can communicate with other hosts

7.HOSTING

=> Web hosting is a service that stores your website or web application and makes it easily accessible across different devices such as desktop, mobile, and tablets.

8.FTP

=> The File Transfer Protocol (FTP) is a standard communication protocol used for the transfer of computer files from a server to a client on a computer network.

9.PATH

=> a path defines the location of a file or folder in a computer's file system.

10.EXTENSION

=> A set of characters at the end of a filename, usually separated by a dot, that indicates the file's format or type.

11.EDITOR /VSCODE

=> An editor, in the context of software development, is a tool used by programmers and developers to create, modify, and manage source code for various programming languages.

12.URL/URI/URN

=> URL- Uniform Resource Locator
=> URI- Uniform Resource Identifier
=> URN- Uniform Resorce Name

13.ADDRESS BAR

=> An address bar is a component of an Internet browser which is used to input and show the address of a website.

14.TITLE BAR

=> a horizontal bar at the top of a window, bearing the name of the program and typically the name of the currently active document.
